# sparkar-pixelate-shader
> simple script-only pixelate shader with Facebook SparkAR.

![preview](./preview.jpg)

## setup

* install deps `npm i`
* webpack watching `npm run dev`
* webpack build `npm run build`

source: [script](./src/index.js)
